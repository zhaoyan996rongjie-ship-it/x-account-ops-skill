# Content Planning

## Weekly Content Cadence

| Day | Content Type | Topic | Format |
|-----|-------------|-------|--------|
| Mon | Experiment Post | Weekly token usage stats | Single post + screenshot |
| Tue | Engagement | Community interaction | Likes + replies |
| Wed | Deep Dive | Tool comparison | Thread (3-5 posts) |
| Thu | Engagement | Community interaction | Likes + replies |
| Fri | Tip/How-to | Prompt engineering tip | Single post |
| Sat | Personal Log | Experiment diary | Single post |
| Sun | Weekly Summary | Metrics + learnings | Thread |

## Content Pillars
Source from Grok's recommended four pillars:
1. **Data Comparison** — Token usage, cost analysis, efficiency metrics
2. **Prompt & Workflow** — Specific prompts, use case walkthroughs
3. **Cost Optimization** — How to reduce token consumption, caching tips
4. **Experiment Logs** — Daily/weekly findings, personal insights

## Thread Structure Template
1. Hook question/observation
2. Data/evidence (screenshots, metrics)
3. Analysis/reasons
4. Practical takeaway
5. Engagement question + hashtags
